<!-- GETTING STARTED -->
## Getting Started

Make sure Node.js is installed in your OS.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/TamerTawfik/bookreviewers.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Add your firebaseConfig secret keys
    ```sh
   lib/firebase.js
   ```
4. start the local server
   ```sh
   npm run dev
   ```
